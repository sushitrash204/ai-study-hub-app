**Aivora App** là ứng dụng di động đa nền tảng (iOS & Android) giúp sinh viên có thể học tập mọi lúc, mọi nơi. Ứng dụng tập trung vào tính tương tác cao, thông báo tức thời và sự hỗ trợ mạnh mẽ từ trí tuệ nhân tạo.

---

## 🚀 Tính năng nổi bật

*   **Mobile Learning:** Học tập và làm bài tập ngay trên điện thoại với giao diện tối ưu.
*   **AI Tutor:** Chat trực tiếp với trợ lý AI, chụp ảnh bài tập để nhận lời giải (Coming soon).
*   **Push Notifications:** Nhận thông báo nhắc nhở học tập, tin nhắn nhóm ngay cả khi tắt ứng dụng.
*   **Offline Support:** Truy cập một số tài liệu học tập ngay cả khi không có mạng.
*   **Seamless Integration:** Đồng bộ hoàn toàn dữ liệu với tài khoản trên phiên bản Web.

---

## 💻 Công nghệ sử dụng

*   **Framework:** Expo SDK 54+ (React Native)
*   **State Management:** Zustand
*   **Deep Linking:** Hỗ trợ mở app từ các liên kết thông báo.
*   **Local Storage:** AsyncStorage để lưu trữ cache và session.
*   **Animations:** React Native Reanimated cho trải nghiệm mượt mà.

---

## 🛠️ Cài đặt và Chạy thử

1. **Clone dự án:**
   ```bash
   git clone https://github.com/sushitrash204/ai-study-hub-app.git
   ```

2. **Cài đặt dependencies:**
   ```bash
   npm install
   ```

3. **Cấu hình biến môi trường:**
   Tạo file `.env` và điền địa chỉ API của bạn:
   ```env
   EXPO_PUBLIC_API_URL=http://your-ip-address:3000/api
   ```

4. **Chạy ứng dụng:**
   ```bash
   npx expo start
   ```
   *Quét mã QR bằng ứng dụng Expo Go trên điện thoại để xem trực tiếp.*
